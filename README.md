# dskpatch
Norton's dskpatch program

## Requirements

  - MAKE (introduced in chapter 15)
  - MASM
  - LINK.EXE
  - EXE2BIN.EXE
  
 
 ## Chapters
 
 Norton starts writing DSKPATCH.COM in chapter 9 (p. 99).
